# 🐧 Club Penguin Homepage UI Clone

## 📌 About the Project

This project is a **UI clone of the Club Penguin homepage**, built as a practice to improve my skills in **HTML and CSS**, focusing on real-world layout recreation.

The goal was to replicate the visual structure and responsiveness of the original website while applying modern CSS techniques and best practices.

---

## 🚀 Technologies Used

* HTML5
* CSS3
* Flexbox
* CSS Grid
* Responsive Design
* CSS Clamp

---

## 🎯 Features

* Responsive layout for different screen sizes
* Navigation bar with interactive hover effects
* Styled buttons and UI components
* Background video integration
* Optimized images with lazy loading

---

## 🧠 What I Learned

* How to recreate real-world UI from an existing website
* Advanced positioning techniques (`absolute`, `relative`, layering with `z-index`)
* Building responsive layouts using media queries and `clamp()`
* Structuring CSS into **base, layout, components, and media queries**
* Improving accessibility with `alt` attributes and `aria-label`
* Performance basics like lazy loading images and controlling video preload
